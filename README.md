OS: Linux (Developed on Debian)

OS Dependencies:
	nikto
	nmap
	xmlstarlet
	curl
	hydra
	nodejs
	npm [or yarn]

Installation:
	git clone https://github.com/bcapptain/webkiddy.git
	cd webkiddy
	npm [or yarn] install

	Scripts from /scripts need to be marked executable and copied/symlinked to /usr/bin/*
